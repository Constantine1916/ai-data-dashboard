# Web Application

这是 AI Data Dashboard 的 Next.js 前端应用。

## 技术栈

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## 开发

```bash
# 从根目录运行
npm run dev

# 或者从 apps/web 目录运行
cd apps/web
npm run dev
```

## 项目结构

```
apps/web/
├── app/              # Next.js App Router
│   ├── layout.tsx    # 根布局
│   ├── page.tsx      # 首页
│   └── globals.css   # 全局样式
├── components/        # React 组件（待创建）
├── lib/              # 工具函数（待创建）
└── public/           # 静态资源（待创建）
```
