# AI Data Dashboard

一个基于 Vite + Vue 3 的 AI 数据仪表板项目。

## 技术栈

- Vue 3
- Vite
- JavaScript

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
ai-data-dashboard/
├── src/
│   ├── App.vue      # 主应用组件
│   ├── main.js      # 应用入口
│   └── style.css    # 全局样式
├── index.html       # HTML 模板
├── vite.config.js   # Vite 配置
└── package.json     # 项目配置
```
